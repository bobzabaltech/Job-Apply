# 2. Dataset Limitations

The dataset represents LinkedIn search results obtained using targeted role keywords.

It provides evidence about:

* employer demand signals;
* terminology used by the market;
* role convergence;
* geographic distribution;
* recurring organizations.

It does not provide reliable information about:

* complete market size;
* total vacancies;
* compensation levels;
* complete technical requirements;
* hiring probability.

Therefore, conclusions must be interpreted as strategic signals rather than exact market statistics.

## 4.2 Manufacturing Systems

Manufacturing Systems roles represent the connection between:

* production processes;
* enterprise systems;
* operational technology;
* software applications.

Typical responsibilities:

* MES implementation;
* manufacturing application development;
* production workflow optimization;
* system integration.

This area has strong alignment with the candidate's combination of:

* manufacturing experience;
* software development;
* data systems.

## 4.3 Industrial Data

Industrial Data roles differ from generic Data Engineering roles.

The relevant problem is not only moving data.

The objective is creating value from:

* machine data;
* production data;
* quality measurements;
* process information.

This is strongly aligned with the candidate's case studies.
