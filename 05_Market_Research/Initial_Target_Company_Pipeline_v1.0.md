# Initial Target Company Pipeline v1.0

## Purpose

Define the first controlled cohort of target companies for validating Julian Cabrera Marceglia's international market positioning.

This pipeline is not designed for mass application.

The objective is to test which market segment generates the strongest response:

* Manufacturing Systems.
* Digital Manufacturing.
* Industrial Data / Manufacturing Intelligence.

---

# Validation Strategy

Initial cohort:

15 companies.

Distribution:

| Segment                                        | Companies |
| ---------------------------------------------- | --------: |
| Industrial Software / Manufacturing Technology |         5 |
| Automotive OEM / Tier 1                        |         5 |
| Digital Manufacturing Consulting               |         5 |

# Company Qualification Before Application

Before applying, verify:

* Open position exists.
* Role matches target positioning.
* Salary potential is acceptable.
* Location is compatible with relocation strategy.
* LinkedIn profile supports the application.
* Relevant case study can be referenced.

---

# Initial Application Goal

First wave:

10 applications.

Not 15.

The remaining companies remain in reserve.

Reason:

The first objective is learning.
