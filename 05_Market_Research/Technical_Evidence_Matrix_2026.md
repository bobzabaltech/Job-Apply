# Technical Evidence Matrix 2026

## Purpose

This document establishes the technical evidence supporting the candidate's professional positioning.

Unlike a skills inventory, this matrix only includes capabilities that can be demonstrated through:

* Professional experience.
* Source code.
* Software architecture.
* Case studies.
* Public portfolio.
* Technical documentation.

Its purpose is to provide a single source of truth for:

* CV development.
* LinkedIn optimization.
* Interview preparation.
* Recruiter conversations.
* Salary negotiation.
