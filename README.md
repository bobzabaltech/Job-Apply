# Decision Framework

Prioritize work according to:

```
ROI =
(Expected impact × Probability of success)
/
Required effort
```

High ROI activities have priority over easy tasks.

Examples:

High ROI:

* Improving market positioning.
* Creating targeted CV variants.
* Optimizing LinkedIn discoverability.
* Building employer targeting strategy.

Lower ROI:

* Minor formatting changes.
* Additional certifications without market evidence.
* Generic portfolio additions.
